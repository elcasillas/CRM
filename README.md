# CRM

A Customer Relationship Management (CRM) application powered by AI.

## Overview

This project is an AI-powered CRM system designed to help manage customer relationships, track interactions, and streamline sales workflows.

## Features

- Customer contact management
- Interaction and activity tracking
- Sales pipeline management
- AI-assisted insights and automation

## Getting Started

### Prerequisites

- Python 3.10+
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/elcasillas/CRM.git
   cd CRM
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Configure environment variables:
   ```bash
   cp .env.example .env
   # Edit .env with your settings
   ```

4. Run the application:
   ```bash
   python main.py
   ```
